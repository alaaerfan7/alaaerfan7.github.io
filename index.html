Featured project

Dynamic ARP cache poisoning detection & prevention

MSc dissertation project: building and testing a system to detect ARP spoofing and automatically enforce mitigation through an SDN controller, while keeping the network usable.

The goal of this project was to go beyond packet captures and create a working defence: a system that could recognise ARP cache poisoning in real time and apply the right countermeasures without breaking normal traffic.

  • Used Python and Scapy to inspect ARP traffic and flag suspicious IP‚ÄìMAC mappings.
  • Integrated with the Ryu SDN controller and Open vSwitch to push dynamic flow rules.
  • Designed the system as a hybrid ARP proxy and detection engine, so the network could continue operating even when attacks were detected.
  • Included a custom ARP attack tool to test edge cases and validate detection coverage.
  • Linked with a DHCP server so both static and dynamic hosts were tracked consistently.
