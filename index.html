Featured project

ARP cache poisoning detection & prevention

MSc dissertation project: building and evaluating a system to detect ARP spoofing in virtual networks and automatically push mitigation rules through an SDN controller.

The project combines low-level packet inspection with controller-level decisions. Rather than only logging suspicious traffic, it updates the network‚Äôs behaviour in real time when an attack is detected.

  • Monitors ARP traffic with Scapy and flags suspicious IP‚ÄìMAC changes.
  • Communicates with the Ryu SDN controller to install or update flow rules.
  • Works as a hybrid ARP proxy / watchdog so normal communication can continue.
  • Includes a custom ARP attack tool to test detection coverage and edge cases.
  • Evaluated with different tools and scenarios in an Open vSwitch lab.
